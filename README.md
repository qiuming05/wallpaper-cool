## 壁纸酷 基于uniapp开发的壁纸程序

### 本项目仅供学习交流，禁止非法商业用途

### 数据均源于 wallhaven.cc 

### 功能说明
* 图片：详情、筛选、查看缩放
* 搜索：但尽量使用英文搜索
* 下载：但不支持间断下载
* 数据缓存：收藏记录、下载记录、预览记录、清除缓存
